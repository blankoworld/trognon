# Trognon

Un lanceur de commandes personnalisées pour votre projet de développement.

Exemple : 

```bash
# Affiche l'aide
./agnes help

# Installe votre projet (vous devez définir ce que `install` fait)
./agnes install

# créé un dump de la base de données
./agnes dump
```
